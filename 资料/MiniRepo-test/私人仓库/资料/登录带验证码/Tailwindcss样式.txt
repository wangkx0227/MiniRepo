# Tailwind CSS 响应式操作全指南

Tailwind CSS 通过基于类前缀的方式实现响应式设计，无需手写任何 `@media` 查询，非常适合组件化前端开发和响应式 Web 页面开发。

---

## 一：默认响应式断点

| 前缀    | 最小屏幕宽度 (以上) | 选用场景       |
| ----- | ----------- | ---------- |
| `sm`  | `640px`     | 手机横屏 / 小平板 |
| `md`  | `768px`     | 平板 / iPad  |
| `lg`  | `1024px`    | 简单桌面端      |
| `xl`  | `1280px`    | 大型显示器      |
| `2xl` | `1536px`    | 超大显示器      |

---

## 二：基本用法

### 格式： `前缀:类名`

```html
<p class="text-sm sm:text-base md:text-lg lg:text-xl">
  我会随幕带变大
</p>
```

* `<640px` 时是 `text-sm`
* `≥640px` 时是 `text-base`
* `≥768px` 时是 `text-lg`
* `≥1024px` 时是 `text-xl`

---

## 三：常用响应式操作类型

### 1. 宽度和幅度

```html
<div class="w-full sm:max-w-sm md:max-w-md lg:max-w-lg">
  响应式宽度
</div>
```

### 2. 排列方向

```html
<div class="flex flex-col md:flex-row">
  <div>项目 1</div>
  <div>项目 2</div>
</div>
```

### 3. 隐藏和显示

```html
<div class="block md:hidden">小屏可见</div>
<div class="hidden md:block">中屏及以上可见</div>
```

### 4. 间距 / 内边距

```html
<div class="p-4 sm:p-6 md:p-8">响应式 padding</div>
<div class="space-y-2 md:space-y-4">组件缓冲间距</div>
```

---

## 四：应用场景总结

| 场景      | 操作推荐                                    |
| ------- | --------------------------------------- |
| 响应式宽度   | `w-full sm:max-w-sm md:max-w-md`        |
| 弹性排列    | `flex-col md:flex-row`                  |
| 显示/隐藏   | `hidden md:block` / `block md:hidden`   |
| 间距切换    | `space-y-2 sm:space-y-4`                |
| 按钮/文字大小 | `text-sm md:text-base` / `px-4 md:px-6` |

---

## 五：Tailwind CDN 快捷引入 (HTML)

```html
<!DOCTYPE html>
<html lang="zh">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <script src="https://cdn.tailwindcss.com"></script>
  <title>响应式示例</title>
</head>
<body class="bg-gray-100 flex items-center justify-center min-h-screen">
  <div class="bg-white p-6 rounded shadow w-full max-w-sm sm:p-8">
    <h1 class="text-lg sm:text-xl font-bold text-center">响应式登录框</h1>
    <form class="mt-4 space-y-4">
      <input class="w-full border p-2 rounded" placeholder="用户名" />
      <input class="w-full border p-2 rounded" placeholder="密码" type="password" />
      <button class="w-full bg-blue-600 text-white p-2 rounded hover:bg-blue-700">登录</button>
    </form>
  </div>
</body>
</html>
```
